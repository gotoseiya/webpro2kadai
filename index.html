  • The head element

    • (1 )head要素のコンテンツであり,body要素のコンテンツと違ってheadのコンテンツはページに表示されない。bodyタグの前,htmlタグの後に書くものであり文書のヘッダ部分を指定するタグとなっている。 

     (2-1) <head>
        <meta charset="utf-8">
        <title>My test page
    </head>
    
    (2-2)
    <head>
        <meta http-equiv="content-type" content="text/html; charset=utf-8">
        <meta name="description" content="">
        <meta name="keywords" content="">
        <title>～<title>
        <link href="style.css" rel="stylesheet" type="text/css" />
        </head>

            <head>
                <title>文書のタイトル</title>
                </head>

  • The style element

    • (1)styleタグとは,HTMLのタグの種類でありhead内に記述することによりタグの開始とタグの終了の間にCSSを記述することができます.style属性とはHTMLのタグに付与できる属性であり,HTMLタグの中にCSSを記述する.

    (2-1)
            <style>
                body {
                    background-color: #00ff00
                }
         
                .container {
                    width: 90%;
                    text-align: center;
                }
         
                .first-sentence {
                    font-weight: bold;
                }
            </style>
